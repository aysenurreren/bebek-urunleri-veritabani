# Bebek Ürünleri Veritabanı

Bu proje, bir bebek ürünleri satış sitesinin veritabanı tasarımını içermektedir.

## Proje Kapsamı
- Ürünler, müşteriler, siparişler tabloları
- Tablolar arası ilişkiler (PRIMARY KEY, FOREIGN KEY)
- Örnek veri girişleri

## Kullanılan Teknolojiler
- SQL (SQL Server)

## Açıklama
Projede temel veritabanı tasarımı uygulanmıştır.
